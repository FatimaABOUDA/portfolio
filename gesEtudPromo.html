Système de Gestion des Étudiants et des Promotions

Classe Etudiant

Attributs :

  • $Code
  • $Nom
  • $Prenom
  • $laPromo (de type Promotion)

Méthodes :

  • __construct() : Constructeur de la classe.
  • Accesseurs : getCode(), getNom(), getPrenom(), getPromo().
  • Mutateurs : setCode(), setNom(), setPrenom(), setPromo().
  • addEtudiant() : Ajoute un étudiant à la base de données.
  • getEtudiants() : Récupère tous les étudiants depuis la base de données.
  • getUnEtudiant(string $code) : Récupère un étudiant spécifique par code.

Classe Promotion

Attributs :

  • $Code
  • $Nom
  • $etudiantCollection (un tableau d'étudiants)

Méthodes :

  • __construct() : Constructeur de la classe.
  • Accesseurs : getCode(), getNom().
  • Mutateurs : setCode(), setNom().
  • getUnePromo(string $code) : Récupère une promotion avec ses étudiants depuis la base de données.
  • getEtudiantCollection() : Récupère tous les étudiants depuis la base de données.

Connexion à la Base de Données

Un fichier config/dataBase.php est inclus pour établir la connexion à la base de données MySQL.
