Création des sites WEB

Chateau9 est une entreprise qui propose des services de création de sites web, de référencement et de marketing digital. Elle est située à Chateauneuf la fôret.

Le site web est un outil de communication indispensable pour une entreprise. Il permet de présenter l’entreprise, ses produits et services, de communiquer sur ses actualités, de générer des contacts et des ventes. Il est donc important de bien le concevoir et de le faire évoluer régulièrement.

Objectifs :

  • Créer un site web pour l’entreprise Chateau9
  • Présenter l’entreprise, ses services et ses réalisations
  • Permettre aux clients de contacter l’entreprise

Contraintes :

  • Le site doit être responsive
  • Le site doit être optimisé pour le référencement naturel
  • Le site doit être sécurisé
  • Le site doit être facilement administrable

Technologies utilisées :

  • HTML5
  • CSS3
  • JavaScript
  • PHP
  • MySQL
  • Bootstrap
  • Git
  • GitHub
  • Visual Studio Code
  • PhpMyAdmin
  • FileZilla
  • WampServer

Les étapes de réalisation :

  1. Création du cahier des charges
  2. Création du design
  3. Création de la base de données
  4. Création du site web
  5. Tests et validation
  6. Déploiement

Le cahier des charges :

Le cahier des charges est un document qui définit les besoins et les attentes du client. Il permet de définir les objectifs du projet et de déterminer les contraintes techniques et fonctionnelles.

Le design :

Le design est la partie graphique du site web. Il permet de définir l’identité visuelle du site et de créer une expérience utilisateur agréable.

La base de données :

La base de données est un ensemble de données structurées. Elle permet de stocker et de gérer les informations du site web.

Le site web :

Le site web est la partie visible du projet. Il permet de présenter l’entreprise, ses services et ses réalisations. Il permet également aux clients de contacter l’entreprise.

Les tests et la validation :

Les tests permettent de vérifier le bon fonctionnement du site web. Ils permettent également de détecter et de corriger les éventuelles erreurs.

Le déploiement :

Le déploiement est la mise en ligne du site web. Il permet de rendre le site accessible aux utilisateurs sur internet.

Conclusion :

Le stage s’est bien déroulé. J’ai réussi à atteindre les objectifs demandés. J’ai appris à travailler en autonomie. J’ai également appris à utiliser de nouvelles technologies.
