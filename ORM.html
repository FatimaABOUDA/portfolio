ORM - Gestion des Familles et Produits

Classe : Famille

  • Propriétés :
    • codeFamille
    • nomFamille
    • AlleeStockage
    • produits
    • laFamille (statique)
  • Constructeur :
    • Initialise les propriétés de la classe et crée une liste vide pour les produits.
  • Méthode CreateFamille :
    • Connecte à la base de données MySQL.
    • Exécute une requête SQL d'insertion pour ajouter une nouvelle famille avec des paramètres spécifiés.
  • Méthodes findOneFamille et findAllFamille :
    • findOneFamille : Recherche une famille par code dans la base de données et renvoie une instance de Famille.
    • findAllFamille : Récupère toutes les familles de la base de données et renvoie une liste d'instances de Famille.

Classe : Produit

  • Propriétés :
    • codeProduit
    • nomProduit
    • laFamille
  • Constructeur :
    • Initialise les propriétés de la classe.
  • Méthode findAllProduit :
    • Recherche tous les produits de la base de données et renvoie une liste d'instances de Produit.
