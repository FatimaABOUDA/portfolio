Méthodes :

Fonctionnalités Principales

  1. Formulaire d'Inscription

    • Un formulaire interactif permet aux utilisateurs de s'inscrire à la compétition.
    • Collecte des informations telles que le nom, le prénom, l'e-mail, l'établissement, la ville, le département, la tranche d'âge, les loisirs préférés, et une description de projet web.

  2. Connexion à la Base de Données

    • Établissement d'une connexion sécurisée à une base de données MySQL grâce à la fonction connexionPDO().
    • Les informations de connexion sont encapsulées dans cette fonction, améliorant la sécurité.

  3. Traitement Sécurisé des Données

    • Utilisation de requêtes SQL préparées pour insérer les données du formulaire dans la table "participant".
    • Prévention des attaques par injection SQL grâce à la liaison de valeurs de formulaire aux paramètres de requête.

  4. Listes Déroulantes Dynamiques

    • Génération dynamique de la liste des départements à partir des données stockées dans la base de données.
    • Utilisation de cases à cocher pour permettre aux participants de sélectionner leurs loisirs préférés.

  5. Feedback Utilisateur

    • Après soumission du formulaire, un message de confirmation est affiché, assurant une expérience utilisateur claire.
