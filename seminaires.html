Fonctions pour charger et sauvegarder des données JSON :

chargeJSONseminaire():
Charge les données du fichier JSON 'data/seminaire.json' et les renvoie sous forme de tableau associatif.
chargeJSONprofessions():
Charge les données du fichier JSON 'data/professions.json' et les renvoie.
sauveJSONseminaire($doc):
Sauvegarde les données passées en argument dans le fichier JSON 'data/seminaire.json'.
chargeJSONadmin():
Charge les données du fichier JSON 'data/admin.json' et les renvoie.

Fonctions pour obtenir des informations sur le séminaire :

donnerIntituleSeminaire():
Retourne l'intitulé du séminaire en extrayant cette information du fichier JSON.
donnerLesHeuresCreneaux():
Retourne la liste de toutes les heures de début des créneaux du séminaire.
donnerLesConferences($heure):
Retourne toutes les conférences commençant à l'heure spécifiée.

Fonctions pour gérer les participants et leurs inscriptions :

sauverDonneesInscription($nom, $prenom, $mail, $ville, $profession):
Sauvegarde les informations d'un visiteur (nom, prénom, etc.) dans le fichier JSON 'data/seminaire.json'.
donnerLesProfessions():
Retourne toutes les professions possibles à partir du fichier JSON 'data/professions.json'.
estInscrit():
Vérifie si un visiteur est déjà inscrit en vérifiant l'existence de variables de session.
enregistre($lesChoix):
Enregistre un participant et ses choix de conférences dans le fichier JSON 'data/seminaire.json'.

Fonctions pour obtenir des informations sur les conférences :

donnerToutesLesConferences():
Retourne toutes les conférences sous forme d'un tableau avec des informations détaillées.
donnerParticipants($numConference):
Retourne tous les participants inscrits à une conférence spécifiée par son numéro.

Fonctions de connexion et vérification d'administrateur :

verifier($login, $mdp):
Vérifie le login et le mot de passe pour autoriser l'accès en tant qu'administrateur.
estAdmin():
Vérifie si le visiteur est un administrateur connecté et autorisé à visualiser les inscriptions.

Fonctions pour vérifier les données d'inscription :

verifierDonneesInscription($nom, $prenom, $mail, $ville):
Vérifie les données saisies lors de l'inscription.

Fonctions liées aux erreurs :

ajouterErreur($msg):
Ajoute un message d'erreur à un tableau.
donnerNbErreurs():
Retourne le nombre de messages d'erreurs.
afficherErreurs():
Affiche tous les messages d'erreur.

Fonction de contrôleur principal :

controleurPrincipal($action):
Détermine le fichier à inclure en fonction de l'action demandée.

Appel du contrôleur principal avec gestion des actions.
